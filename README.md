# orca.nvim (WIP)
New nord inspired neovim colorscheme.
Feel free to try it out.
Feedback is very welcome!

### Features
- TreeSitter
- nvim-cmp
- Nvim-Tree

### Install
```lua
use({
	"devnnys/orca.nvim",
	config = function()
		require("orca")
	end,
})
```
